# Security Policy

## Supported Versions

coop is pre-1.0. Only the latest available beta version receives security
fixes. Older beta versions and arbitrary development snapshots are not
supported; a fix may require upgrading to a newer beta with incompatible CLI
or configuration changes.

## Reporting a Vulnerability

Report suspected vulnerabilities through GitHub's private vulnerability
reporting route:

<https://github.com/sarcasticbird/coop/security/advisories/new>

Include the affected version or commit, reproduction steps, impact, and any
known mitigations. Do not open a public issue for an undisclosed vulnerability.
Private vulnerability reporting is enabled for this public repository and is
available to any GitHub user.
